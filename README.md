# Vivaldi Libadwaita (GNOME Style)

CSS customizado para integrar o Vivaldi ao ecossistema GNOME. Focado em minimalismo, consistência visual e na estética do **Libadwaita**.

### 🛠 O que foi modificado:

* **Abas (Floating Tabs):** Formato de pílula arredondada seguindo o padrão visual do GNOME 45+.
* **Speed Dial:** Ícones reduzidos e centralizados com padding generoso para evitar poluição visual.
* **View Switcher:** Barra de navegação interna (Favoritos/Histórico) redesenhada com botões discretos e centralizados no topo.
* **Paleta Adwaita:** Uso das cores oficiais (`#1D1D20`, `#2E2E32`, `#343436`) para total integração com o tema escuro do sistema.
* **Layout Nativo:** Menu hambúrguer movido para a direita e remoção dos botões de minimizar/maximizar (padrão GNOME).
* **Scrollbars:** Barras de rolagem finas e arredondadas que aparecem apenas sob demanda.

---

### 🚀 Instalação do CSS:

1.  Abra `vivaldi://experiments` e ative a opção **"Allow CSS modifications"**.
2.  Vá em **Configurações > Aparência** e, na seção **Modificações para a Interface**, selecione a pasta onde você salvou o arquivo `gnome-theme.css`.
3.  Reinicie o navegador.

### 📦 Como importar o Tema (`.zip`):

Este repositório inclui um arquivo `.zip` que contém as cores e ícones customizados do Speed Dial:

1.  Vá em **Configurações > Temas**.
2.  Clique no ícone de **"+" (Adicionar Tema)** e selecione **Instalar Tema**.
3.  Escolha o arquivo `.zip` baixado deste repositório.
4.  Após aplicar, certifique-se de que o CSS externo ainda está ativo no passo da Instalação acima.
